# Anime Catalog (SaaS-ready, no billing)

## Требования
- Node.js 20+

## Быстрый старт
1. Установить зависимости:
   ```bash
   npm install
   ```
2. (Опционально) создать `.env` на базе `.env.example`.
3. Запуск dev (API + Vue):
   ```bash
   npm run dev
   ```
4. Открыть:
   - `http://127.0.0.1:8008`

## Production
```bash
npm run build
npm start
```

API также слушает `127.0.0.1:8008`.
